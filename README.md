# WyxSync

Aplicativo de gestão fitness que integra acompanhamento de treinos, monitoramento nutricional e inteligência artificial para auxiliar usuários na construção de hábitos saudáveis e acompanhamento da evolução física.

🌐 Frontend: wyxsync-web.vercel.app

⚙️ Backend API: Em ambiente privado

📱 Status: MVP funcional

## Visão Geral

O WyxSync foi desenvolvido com o objetivo de centralizar em uma única plataforma os principais recursos utilizados por praticantes de atividade física, permitindo o gerenciamento de treinos, controle nutricional e interação com um assistente virtual baseado em IA.

O sistema foi projetado para oferecer uma experiência moderna, intuitiva e escalável, utilizando uma arquitetura dividida entre fronten
d, backend e serviços externos.

## Principais Funcionalidades

### Autenticação

* Cadastro de usuários
* Login com e-mail e senha
* Autenticação com Google
* Controle de sessão via JWT

### Gestão de Treinos

* Agenda semanal de treinos
* Filtros por objetivo
* Treinos personalizados
* Controle de progresso

### Nutrição

* Controle de calorias
* Acompanhamento de macronutrientes
* Registro de refeições
* Upload de imagens para análise alimentar

### Inteligência Artificial

* Assistente virtual Wyx
* Sugestões de treinos
* Sugestões nutricionais
* Interação por texto e voz

## Tecnologias Utilizadas

### Frontend

* React
* JavaScript
* Vercel

### Backend

* ASP.NET Core
* Entity Framework Core
* JWT Authentication

### Banco de Dados

* PostgreSQL

### Serviços Externos

* OpenAI
* Google Authentication
